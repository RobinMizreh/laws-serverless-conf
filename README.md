# LAWS - Un site dynamique en serverless c'est possible

## Comment recréer le projet chez vous

 * Suivre toute la procédure documentation-step-by-step
 * Les static sont dans le dossier site-static
 * Le dossier awsConfig contient le code des lambda, la donnée initiale pour dynamoDB et la policy IAM (ne pas oublier de changer l'accountID, voir la région si vous n'ètes pas en Irelande)
 * Dans les static le fichier js contenant la configurations à modifier se trouve dans js/config.js
 * N'oublier pas de mettre en public les fichier sur S3 afin que le site static fonctionne
